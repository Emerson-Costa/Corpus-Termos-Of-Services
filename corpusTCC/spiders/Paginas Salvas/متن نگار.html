Privacy Policy

Matnnegar users privacy is very important for us. We exchange information and data in a secure context and if this information is used, the data will definitely be analyzed and processed without the names of its holders. This information will be used to improve the provision of services of Matnnegar

Your information is divided into two personal and user data. Personal information is information that can be used to identify a user that you've sent directly to us.

Personal information includes items such as email and mobile numbers that will not be publicly displayed.  User data is information through which personality and data are not possible and identifiable. These statistics help Matnnegar to better understand its users and its software needs, as well as in Matnnegar updates.

User data such as model, brand, manufacturer, operating system version, profile page, status and type of network and Matnnegar Theme and Matnnegar selected language are sent to server.
